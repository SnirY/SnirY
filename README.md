## 🔭 Selected Projects

### 🏥 [NICU Infant Length Measurement](https://github.com/SnirY/Infant-Length-Measurement-System)
Computer vision prototype for non-invasive infant length estimation from a single top-down RGB image using **YOLOv8, OpenPose, and OpenCV**.  
Designed around a NICU-oriented measurement workflow, combining object detection, pose-based analysis, and image-based calibration.  
`Python` `YOLOv8` `OpenPose` `OpenCV`

### ⚽ Football Match Prediction ML Pipeline
End-to-end machine learning pipeline for predicting football match outcomes using historical data, feature engineering, and structured storage.  
Includes data collection, feature engineering, model training, and backtesting workflows built for practical evaluation and decision support.  
`Python` `XGBoost` `PostgreSQL` `REST APIs`
